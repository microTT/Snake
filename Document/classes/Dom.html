editClassName

(
  • element
  • option
  • classString
)
HTMLElement | Boolean

Provided by the Util module.

Defined in snake.js:211

编辑DOM对象的className。使用最新的HTML5的classList对象操作对象className,若浏览器不支持则采用“手动”处理的方式。 动态定义函数，第一次运行时进行浏览器能力检测并重新定义函数。

Parameters:

  • element HTMLElement

    要进行类名操作的DOM对象

  • option String

    操作类型，只能为["add", "remove", "contains", "toggle"]中的一个，若不为要求字符串其中之一则抛出错误。

  • classString String

    要进行编辑的类名

Returns:

HTMLElement | Boolean:

若option为"contains",则返回一个Boolean表示检测包含的结果，若为其他操作则返回类名操作的DOM对象
